theme property

ThemeData theme

The data from the closest Theme instance that encloses the given context.

Typical usage is as follows:

return Text(
  "Hello World!",
  style: context.theme.textTheme.headline6,
);

Implementation

ThemeData get theme {
  return Theme.of(this);
}
